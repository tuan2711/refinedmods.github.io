Storage Disk

  1. Refined Storage
  2. Wiki
  3. Storage Disk

A Storage Disk is an item where items from the storage network are stored on. It has to be put in a Disk Drive.

The Storage Disk won’t despawn when dropped in the world.

Uncrafting

Sneak and right click while holding the empty Storage Disk to retrieve the various parts back:

  • The corresponding Storage Part
  • Storage Housing
  • Guides
  • Blocks
  • Items
